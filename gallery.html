Build and Conquer 1.16

Remake of Build and Conquer which I made four years ago.

A Base Building Game where you set up infrastructure and conquer other players' land!

Updates since last version:

- New Models

- Huge performance improvements. The code on the old version ran on something like O(N^2 log(N)) with N being the number of buildings. Don't ask me how that happened. Now it runs on O(N), sustaining minimal lag even on older machines.

- A few new mechanics

- New balancing features (Electrical generation is suspended for a minute when a rocket is launched)
